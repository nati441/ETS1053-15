# isupper() string method
The isupper() method in Python is a string method that checks whether all the alphabetic characters in a string are uppercase. It returns True if all letters are uppercase and False otherwise. If the string contains no alphabetic characters, it returns False.
# islower() string method
The islower() method in Python is a string method that checks whether all the alphabetic characters in a string are lowercase. It returns True if all letters are lowercase and False otherwise. If the string contains no alphabetic characters, it returns False.
