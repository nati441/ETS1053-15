# str.capitalize
In Python, the str.capitalize() method returns a copy of a string with the first character capitalized and all other characters converted to lowercase.
# str.swapcase
The str.swapcase() method in Python returns a new string with all uppercase letters converted to lowercase and all lowercase letters converted to uppercase.

# str.find
The str.find() method in Python is used to find the first occurrence of a substring within a string. If the substring is found, it returns the index of the first character of the substring. If the substring is not found, it returns -1.
